## Desenvolvimento

### Setup

- Ter o `Node` instalado
- O projeto utiliza o `yarn` como gerenciador de pacotes
- Baixar dependencias do projeto
  ```bash
  yarn
  ```

### Iniciar projeto
- Start do projeto localmente
  ```
  yarn dev
  ```

### Outros scripts
- build para produção
  ```
  yarn build
  ```
